Telegram-бот для продажи Telegram Stars и TON.

Установка

pip install -r requirements.txt
python bot.py
Настройка

Открой bot.py и измени в блоке НАСТРОЙКИ:

Переменная	Описание
TOKEN	Токен бота от @BotFather
ADMIN_IDS	Список Telegram ID администраторов
SUPPORT_USERNAME	@юзернейм поддержки
CARD_NUMBER	Номер карты для оплаты
CARD_PHONE	Телефон для оплаты
CRYPTO_ADDRESS	TON-адрес для крипто-оплаты
STARS_PRICE_RUB	Цена 1 звезды в рублях
RATES	Курсы валют
