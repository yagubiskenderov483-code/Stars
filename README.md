# 🤖 Telegram Stars Bot

Бот для покупки и передачи звезд в Telegram с функцией создания чеков.

## 📋 Возможности

✅ Веб-кошелек (https://stars-zdgz.onrender.com)
✅ Вывод звезд (требуется Fragment аккаунт)
✅ Создание чеков с фото
✅ Автоскупщик подарков (в разработке)
✅ Система ограничения на создание чеков (21 день или команда /neptunteam)
✅ Админ-панель с логами
✅ Скрытие логов

## 🚀 Установка

1. **Установите зависимости:**
```bash
pip install -r requirements.txt
```

2. **Получите токен бота:**
   - Откройте [@BotFather](https://t.me/BotFather) в Telegram
   - Создайте нового бота командой `/newbot`
   - Скопируйте полученный токен

3. **Настройте бота:**
   - Откройте файл `bot.py`
   - Найдите строку: `BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"`
   - Замените `YOUR_BOT_TOKEN_HERE` на ваш токен
   - Найдите строку: `ADMIN_ID = 123456789`
   - Замените `123456789` на ваш Telegram ID

4. **Узнайте свой Telegram ID:**
   - Откройте [@userinfobot](https://t.me/userinfobot)
   - Отправьте `/start`
   - Скопируйте ваш ID

5. **Запустите бота:**
```bash
python bot.py
```

## 🎮 Команды

- `/start` - Запуск бота и главное меню
- `/neptunteam` - Разрешение на создание чеков (обходит ограничение в 21 день)
- `/admin` - Админ-панель с логами (только для администратора)

## 🔧 Основные функции

### Для пользователей:
- **💳 Веб-Кошелек** - открывает веб-интерфейс
- **🔔 Вывести звезды** - вывод звезд (требует аккаунт Fragment)
- **👛 Кошелек** - просмотр баланса
- **👑 Создать чек** - создание чека с фото (требует прав)

### Создание чека:
1. Нажмите "👑 Создать чек"
2. Введите количество звезд
3. Отправьте фото
4. Получите ID чека

### Ограничения на создание чеков:
- Новые пользователи: нужно подождать 21 день после регистрации
- Обход ограничения: команда `/neptunteam`

### Админ-панель:
- Просмотр логов активности пользователей
- Скрытие логов
- Просмотр скрытых логов

## 📊 База данных

Бот автоматически создает SQLite базу данных `bot_database.db` с таблицами:
- `users` - информация о пользователях
- `checks` - созданные чеки
- `logs` - логи активности

## 🔐 Безопасность

- Админ-панель доступна только владельцу бота (ADMIN_ID)
- Все взаимодействия логируются
- Возможность скрытия чувствительных логов

## 📝 Примечания

- Все функции "в разработке" (Автоскупщик, Магазин, Пополнение) можно доработать по необходимости
- Веб-кошелек подключен через Web App API Telegram
- Бот использует inline-режим для всех кнопок
